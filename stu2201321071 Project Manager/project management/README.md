# Project Management

### Main Goal of this Position
* **Picking the right individual** for the specific task using the method "Six Thinking Hats";
* **Making user stories** based of the users preferences;
* **Choosing the important fuctionalities** form the user stories;
* **Split the tasks** for the roles;
* **using "Kanban" method** objective view of the what we are doing based of 5 stages:

| Stage | Process Step |
| :--- | :--- |
| 1 | User stories; |
| 2 | Task; |
| 3 | In Progress; |
| 4 | Testing; |
| 5 | Done; |
